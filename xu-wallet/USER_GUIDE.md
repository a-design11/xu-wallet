# XU Wallet - User Guide

## 🚀 Getting Started

### First Time Setup

1. **Launch the App**
   - See animated XU logo with "Your Keys. Your Crypto."
   - Click "Continue" to proceed

2. **Choose Wallet Option**
   - **Create Wallet**: Set up a new wallet from scratch
   - **Import Wallet**: Restore existing wallet with seed phrase

### Creating a New Wallet

1. **Set Your PIN**
   - Enter a 6-digit PIN
   - Confirm the PIN by entering it again
   - Optional: Enable biometric authentication

2. **Save Your Seed Phrase**
   - Write down the 12-word recovery phrase
   - Keep it in a safe place (never share it!)
   - Click "Copy to Clipboard" for backup
   - Click "I've Written It Down" to continue

3. **Verify Seed Phrase**
   - Select the correct words in order
   - This ensures you've saved it correctly
   - Click "Verify & Complete"

### Importing an Existing Wallet

1. Select "Import Wallet" on onboarding screen
2. Enter your 12-word seed phrase
3. Click "Import Wallet"
4. Access your wallet immediately

## 💼 Using Your Wallet

### Home Dashboard

**Top Section:**
- Wallet icon and abbreviated address
- Total balance (tap eye icon to hide/show)
- Quick actions: Send & Receive

**Blockchain Selector:**
- Tap tabs to switch between chains:
  - Ethereum
  - Solana
  - BNB Chain
  - Polygon

**Network Toggle:**
- Switch between Mainnet and Testnet

**Token List:**
- View all tokens for selected blockchain
- Tap any token for details
- Click "+ Import Token" to add custom tokens

### Sending Crypto

1. From Home, tap "Send" or select a token and tap "Send"
2. Choose token from dropdown
3. Enter recipient address
4. Enter amount (tap "Max" for full balance)
5. Review gas fee
6. Tap "Review Transaction"
7. Enter PIN to confirm
8. Transaction sent!

### Receiving Crypto

1. Tap "Receive" or QR icon
2. Share QR code or copy address
3. Verify the network matches
4. Share with sender

### Importing Custom Tokens

1. Tap "+ Import Token" on Home
2. Paste contract address
3. Click "Fetch Token Info"
4. Review token details
5. Click "Import Token"
6. Token appears in your list

## 🌐 DApp Browser

### Browsing DApps

1. Tap "Browser" in bottom navigation
2. Browse popular DApps or search
3. Filter by category: All, DeFi, NFT, Gaming
4. Tap any DApp to open

### Connecting to DApps

1. Open a DApp
2. Click "Connect Wallet"
3. Review connection details
4. Read security warning
5. Click "Connect"
6. Your wallet is now connected

### Approving Transactions

1. When DApp requests transaction, review details:
   - From address
   - Action type
   - Gas fee
2. Click "Approve" or "Reject"
3. If approved, enter PIN to confirm
4. Transaction processed

## 🔐 Security Best Practices

### ✅ DO:
- Write down seed phrase on paper
- Store seed phrase in multiple safe locations
- Use a strong PIN
- Enable biometric authentication
- Verify DApp URLs before connecting
- Double-check recipient addresses
- Review all transaction details

### ❌ DON'T:
- Share your seed phrase with anyone
- Store seed phrase digitally (screenshots, cloud)
- Connect to suspicious websites
- Ignore security warnings
- Share your PIN
- Send crypto to unknown addresses

## 🆘 Troubleshooting

### Forgot PIN
- You'll need your seed phrase to restore wallet
- No other recovery method available

### Lost Seed Phrase
- If you have access, export it from Settings
- Without seed phrase, wallet cannot be recovered

### Transaction Stuck
- Check network status
- Verify sufficient balance for gas
- Wait for network confirmation

### Token Not Showing
- Ensure correct blockchain is selected
- Try importing token manually with contract address
- Check network (Mainnet vs Testnet)
